# jpn-latn

日本語をlatin字に變換する. 通常の*ローマ字*でなく, 歷史的かな遣ひ
と漢語に基く獨自の系を用ゐる.

```sh
deno task run -- "これは例文です。巧く變換できるかな？"
-- kore fa rèimbun desu. umaku hènkvàn dekiru ka na?
```

## 槪要

jpn-latnは日本語をlatin字で表記する方法.
漢詞を中古音に基き表記する事で同音詞を區別する.

## 母音

| 段  | latn |
| :-: | :--: |
| あ  | `a`  |
| い  | `i`  |
| う  | `u`  |
| え  | `e`  |
| お  | `o`  |

## 子音

| 行  | latn |
| :-: | :--: |
| あ  |  ``  |
| か  | `k`  |
| が  | `c`  |
| さ  | `s`  |
| ざ  | `z`  |
| た  | `t`  |
| だ  | `d`  |
| な  | `n`  |
| ぱ  | `p`  |
| は  | `f`  |
| ば  | `b`  |
| ま  | `m`  |
| や  | `j`  |
| ら  | `r`  |
| わ  | `v`  |

## 半母音

| 行  |   latn    |
| :-: | :-------: |
| ゃ  | `(C)j(V)` |
| ゎ  | `(C)v(V)` |

## 漢詞

|    由來    |          kana          |      latn       |
| :--------: | :--------------------: | :-------------: |
|  鼻音聲母  | が行, ざ行, だ行, ば行 | `g, nz, nd, mb` |
|  [p]聲母   |          は行          |       `p`       |
|  [ŋ]韻尾   |         い, う         |     `ĩ, ũ`      |
| [m, n]韻尾 |           ん           |     `m, n`      |
|  入聲韻尾  |   き, く, ち, つ, ふ   | `k, k, t, t, p` |
|  [p]韻尾   |           う           |       `p`       |
| 上聲, 去聲 |          母音          | U+0301, U+0300  |

子音尾で終はる音節と母音または半母音で始まる音節が隣接して曖昧な時は,
後者を`q`で始める.

## 活用

活用詞に現れる促音は本來の子音で書く.

## 例

| jpn                          | latn                                        |
| :--------------------------- | :------------------------------------------ |
| 日本語を正しく讀むのは難しい | nitpóngó vo tadasiku jomu no fa mudukasii　 |
| 一萬人の觀客を前に歌った     | itmànnin no kvankjak vo mafe ni utafta      |
| 三回 回って わんと言へ       | samkwai mafarte van to ife                  |
| 山陽道は遠い                 | sanqjagdáu fa tofoi                         |
